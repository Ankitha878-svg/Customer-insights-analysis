# Customer-insights-analysis
Data-driven analysis of customer shopping behavior using Python, SQL and Power BI to uncover purchasing patterns, customer segments, and business insights.

## Overview

This project focuses on analyzing customer shopping behavior to derive meaningful business insights. The workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and interactive dashboard creation.

The goal is to understand customer purchasing patterns, identify trends, and support data-driven decision-making.

## Dataset

The dataset contains customer transaction details such as:

* Customer demographics
* Purchase history
* Product categories
* Discounts and promotions
* Transaction frequency

Data was preprocessed to handle:

* Missing values
* Inconsistent formats
* Duplicate records

## Tools & Technologies Used

* Python (Pandas, NumPy) – Data loading, cleaning, and EDA
* MS SQL Server – Writing and executing analytical queries
* Power BI – Building interactive dashboards and visualizations
* Gamma – Creating presentation (PPT)
* Microsoft Excel – Initial data handling (

## Exploratory Data Analysis (EDA)

Performed detailed EDA to uncover:

* Customer purchase patterns
* Most popular product categories
* Impact of discounts on sales
* Customer segmentation based on behavior

Visualizations helped in identifying trends and anomalies in the dataset.

## Data Cleaning

* Handled missing and null values
* Standardized column names and formats
* Removed duplicates
* Converted data types for better analysis

## SQL Analysis (MS SQL Server)

Executed SQL queries to:

* Identify top-selling products
* Analyze customer segments (new, returning, loyal)
* Calculate discount usage rate
* Extract category-wise insights

## Power BI Dashboard

Developed an interactive dashboard to present:

* Sales trends and KPIs
* Customer segmentation
* Product performance
* Discount impact analysis

The dashboard enables quick and effective business decision-making.

## Results & Insights

* Identified key purchasing trends and customer preferences
* Highlighted top-performing product categories
* Analyzed the effectiveness of discounts
* Segmented customers for targeted strategies

## Conclusion

This project demonstrates end-to-end data analytics skills, from data preprocessing to visualization and reporting, using industry-relevant tools like Python, SQL, and Power BI.
